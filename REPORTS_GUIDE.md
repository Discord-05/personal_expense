# Reports Page Guide

## Overview
The Reports page provides advanced analytics and insights into your spending patterns with interactive charts, detailed data tables, and export capabilities. Analyze expenses across custom date ranges and export data for external analysis.

## Features

### 1. Date Range Selection
- **8 Preset Ranges**: Quick access to common time periods
- **Custom Range**: Select any start and end date
- **Dynamic Filtering**: All charts and tables update automatically
- **Range Display**: Clear indication of selected period

### 2. Visual Analytics
- **Category Pie Chart**: See expense distribution by category
- **Expense Trend Chart**: Track spending over time (line or bar chart)
- **Interactive Charts**: Hover for detailed tooltips
- **Chart Export**: Download charts as PNG images

### 3. Data Tables
- **Category Summary**: Totals, counts, averages, and percentages by category
- **Transaction Details**: Complete list of filtered expenses
- **Sortable Columns**: Multiple sorting options
- **Visual Indicators**: Color-coded bars and icons

### 4. Data Export
- **CSV Export**: Download filtered data for spreadsheet analysis
- **Formatted Data**: Properly escaped for Excel/Google Sheets
- **Named Files**: Automatic filename with date range

### 5. Summary Statistics
- **Total Expenses**: Sum of all expenses in date range
- **Daily Average**: Average spending per day
- **Top Category**: Highest spending category
- **Active Categories**: Number of categories used

## Using the Reports Page

### Analyzing Spending

1. **Select Date Range**
   - Click a preset button (e.g., "Last 30 Days")
   - Or click "Custom Range" and select dates
   - All data updates automatically

2. **Review Statistics**
   - Check total spending for period
   - See daily average to understand patterns
   - Identify top spending category
   - Note number of active categories

3. **Analyze Charts**
   - **Pie Chart**: Which categories consume most budget?
   - **Trend Chart**: When do you spend most?
   - Toggle line/bar view for different perspectives
   - Hover over elements for exact values

4. **Study Data Tables**
   - **Category Summary**: Which category needs attention?
   - Sort by amount to find biggest expenses
   - Check percentages to understand distribution
   - Review transaction counts for frequency

5. **Export Data**
   - Click "Export to CSV" for external analysis
   - File includes all transactions in date range
   - Open in Excel, Google Sheets, or other tools
   - Use for budgeting, tax prep, or record-keeping

### Chart Interpretation

#### Pie Chart Insights
- **Dominant slice**: Your biggest expense category
- **Multiple large slices**: Balanced spending across categories
- **Many small slices**: Diverse spending patterns
- **Single large slice**: May indicate budget imbalance

#### Trend Chart Insights
- **Upward trend**: Increasing spending over time
- **Downward trend**: Successfully reducing expenses
- **Spikes**: Unusual or one-time purchases
- **Flat line**: Consistent spending pattern

### Sorting Data

Use the **Sort by** dropdown on Category Summary:

- **Amount (High to Low)**: See biggest spenders first (default)
- **Amount (Low to High)**: See smallest expenses first
- **Category (A-Z)**: Alphabetical order
- **Percentage**: By share of total spending
- **Transaction Count**: By frequency of use

## Troubleshooting

### No Data Showing
- **Check date range**: May be too narrow or in the future
- **Verify expenses exist**: Add expenses if none recorded
- **Try "All Time"**: See if any data exists
- **Browser console**: Check for JavaScript errors

### Charts Not Rendering
- **Refresh page**: Clear cache and reload
- **Check browser**: Ensure JavaScript enabled
- **Try different browser**: Test compatibility
- **Disable extensions**: Ad blockers may interfere

### Export Not Working
- **Check for data**: Can't export empty results
- **Browser settings**: Allow downloads
- **Pop-up blocker**: May block download
- **File permissions**: Check download folder access

### Slow Performance
- **Large date range**: Try shorter periods
- **Many transactions**: Consider monthly exports
- **Close other tabs**: Free up browser memory
- **Clear browser cache**: Remove old data
